# Luxe Phone Book

## Descripción

Aplicación web de directorio telefónico para resort de lujo.

## Características

- Búsqueda de contactos
- Filtrado por categorías
- Interfaz responsive
- Diseño moderno

## Instalación

1. Clonar repositorio
2. Abrir `index.html` en un navegador

## Tecnologías

- HTML5
- CSS3
- JavaScript Vanilla

## Estructura

```
luxephonebook/
├── public/
│   ├── assets/
│   ├── js/
│   └── pages/
└── README.md
```
